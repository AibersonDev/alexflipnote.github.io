About xelA:

xelA is a bot which it's main duty is to keep your discord server safe. Currently the bot is private and for communities only, however you can request the bot by sending a DM to AlexFlipnote#0021 on Discord. You can find Alex by clicking here

Why xelA:

With xelA, you'll be sure to have cool things such as fun commands for anyone to play around with and some awesome moderator commands which will make your server less messy to work with. The bot provides logs for most of the important things that are needed as "ban, unban, mute, unmute" and more to come.

Terms of Service

By using xelA on your server, you agree with the rules provided for security reasons or/and ensuring that nothing shall go wrong.

[ 1 ] Usage

Using xelA is mainly for making your server have more fun or/and moderate your server to a whole new level. Abusing the bot in any kind as "spam" or any kinds of actions that results to the bot to slow down, will be taken care of by an immediate removal of the bot from any server it servers.

[ 2 ] Logging

To provide the best service possible, xelA must be able to log usage of commands or actions made in the server regarding Audit logs This information will in no way, be abused or used to show negative inpact to any server.

[ 2 . 1 ] Audit Logs

By allowing xelA to see Audit logs, you agree on letting the activity be shared to the bot for both statistic or/and usage to your server after permitting. Any information that is shared to the bot regarding Audit logs will not be shared with anyone and will be deleted after 1 week of being stored on the server.

[ 2 . 2 ] Personal information

xelA does not store any public user's information such as Discord ID, username or discriminator. The bot only stores Discord ID's to the respected bot owner(s) for security reasons, such as command restrictions.

[ 3 ] Announcement

The bot does have a feature where you can use a webhook to post an announcement to a chosen discord server channel. Using this does require webhook credentials, which is stored on the server, encrypted. Abusing the webhook usage will result in an immediate removal of the bot from the abusive server.
